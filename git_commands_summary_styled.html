Підсумок основних команд Git

  • 1. Перевірка поточної гілки

    git branch

  • 2. Перемикання між гілками

    git checkout master
    git checkout main

  • 3. Створення нової гілки

    git checkout -b feature/new-project

  • 4. Додавання файлів до Git

    git add .

  • 5. Зробити коміт

    git commit -m "Додав нові файли для проекту"

  • 6. Перегляд історії комітів

    git log

  • 7. Відправка змін на GitHub

    git push -u origin feature/new-project

  • 8. Перевірка статусу файлів

    git status

  • 9. Злиття гілок

    git merge feature/new-project
